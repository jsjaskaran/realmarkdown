# realmarkdown


----------


Real time markdown conversion along with link saving

### Steps to run
 - clone the repo
 - run `npm install`
 - run `npm start`

#### Side Note:
currently using redis for saving links and data, can also use sharedb along with sharedb-mongo to use mongodb
